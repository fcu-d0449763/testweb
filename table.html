描述性綱目事件條列式
使用者進行且通過帳號密碼驗證之後，點選進入「圓夢計畫頁面」。使用者 + 輸入 + 帳號密碼
驗證伺服器 + 接收 + 帳號密碼
驗證伺服器 + 檢核 + 帳號密碼
系統 + 轉接 + 圓夢計畫頁面
使用者點選進入「新增圓夢計畫」，進入「新增圓夢計畫頁面」。使用者 + 點選 + 按鈕
系統 + 轉接 + 新增圓夢計畫頁面
使用者輸入計畫內容，點選送出，使用者成功送出資料之後，轉跳到詳細圓夢計畫內容。使用者 + 輸入 + 計畫內容
使用者 + 點選 + 送出按鈕
系統 + 接收 + 計畫內容
系統 + 檢核 + 計畫內容
資料庫 + 新增 + 計畫內容
系統 + 轉接 + 計畫詳細頁面
使用者欲修改計畫內容，點選修改圓夢計畫按紐，進入「修改圓夢計畫頁面」。使用者 + 點選 + 修改按紐
系統 + 轉接 + 修改圓夢計畫頁面
使用者輸入欲修改內容，點選送出，使用者成功送出資料之後，轉跳到詳細圓夢計畫內容。使用者 + 輸入 + 計畫內容
使用者 + 點選 + 送出按鈕
系統 + 接收 + 計畫內容
系統 + 檢核 + 計畫內容
資料庫 + 新增 + 計畫內容
系統 + 轉接 + 計畫詳細頁面
